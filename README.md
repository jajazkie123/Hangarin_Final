# Hangarin — To-Do Manager

A Django-based personal task and to-do management web application built as a midterm project.

## Features

- **Dashboard** — Live summary cards (total tasks, completed, pending, in progress, this year, overdue)
- **Full CRUD** — Create, Read, Update, Delete tasks, categories, and priorities
- **Search** — Search tasks by title, description, or category
- **Sorting** — Sort by title, created date, deadline, or status
- **Filtering** — Filter by status, category, and priority
- **Pagination** — All list views are paginated (10 per page)
- **Subtasks** — Add and toggle subtasks per task
- **Notes** — Add notes to any task
- **Admin Panel** — Fully configured Django Admin with search, filters, and custom displays
- **Authentication** — Login/logout with per-user task isolation
